## Co zostało dodane?

### 1. 🎬 AI Video Stories
Przekształcaj posty w video dla Instagram, TikTok i więcej!

### 2. 🚀 Multi-Platform Optimizer  
Jeden post → Optymalizacja dla wszystkich platform jednocześnie!

---

## Jak Używać?

### Video Stories (3 kroki):
1. **Wygeneruj post** w Generator View
2. **Kliknij "Video Story"** (fioletowy przycisk)
3. **Wybierz styl** → Generuj → Gotowe! 🎥

### Multi-Platform (3 kroki):
1. **Wygeneruj post** w Generator View
2. **Przewiń w dół** do Multi-Platform Optimizer
3. **Zaznacz platformy** → Optymalizuj → Kopiuj! 📋

---

## Uruchomienie

```bash
# Terminal 1 - Backend
cd server
npm run dev

# Terminal 2 - Frontend
npm run dev
```

Otwórz: `http://localhost:3000`
